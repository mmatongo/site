# dnlm.pw
Code for [dnlm.pw](https://dnlm.pw), a personal website.

## Technology

- HTML, CSS
- [GoLang](https://golang.org/)

## Development

```
$ go run ./cmd/dnlm
```
or

```
$ go build -o ./bin/web ./cmd/dnlm
$ ./bin/dnlm
```
